API Endpoints

Exercises

GET /exercises Get all exercises with filtering
GET /exercises/{id} Get specific exercise by ID
GET /exercises/search/{query} Search exercises by name
GET /exercises/random Get random exercises

Filtering

GET /exercises/by-muscle/{muscle} Filter by target muscle
GET /exercises/by-bodypart/{part} Filter by body part
GET /exercises/by-equipment/{equipment} Filter by equipment

Reference Data

GET /muscles Get all muscle groups
GET /bodyparts Get all body parts
GET /equipments Get all equipment types
GET /stats API statistics

Code Examples

Get All Exercises

curl "http://localhost:8000/exercises?page=1&limit=10"

Search Exercises

curl "http://localhost:8000/exercises/search/push%20up"

Get Exercises by Muscle

curl "http://localhost:8000/exercises/by-muscle/biceps"

Fetch Exercises with JavaScript

// Get all exercises
const response = await fetch('http://localhost:8000/exercises?limit=10');
const data = await response.json();
console.log(data.exercises);

// Search exercises
const searchResponse = await fetch('http://localhost:8000/exercises/search/squat');
const searchData = await searchResponse.json();
console.log(searchData);

// Get exercises by muscle
const muscleResponse = await fetch('http://localhost:8000/exercises/by-muscle/chest');
const muscleData = await muscleResponse.json();
console.log(muscleData);

Python with Requests

import requests

# Get all exercises
response = requests.get('http://localhost:8000/exercises', params={'limit': 10})
exercises = response.json()
print(exercises['exercises'])

# Search exercises
search_response = requests.get('http://localhost:8000/exercises/search/deadlift')
search_data = search_response.json()
print(search_data)

# Get exercises by equipment
equipment_response = requests.get('http://localhost:8000/exercises/by-equipment/dumbbell')
equipment_data = equipment_response.json()
print(equipment_data)
